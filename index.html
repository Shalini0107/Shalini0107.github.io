Full Stack · Python + React JS + MySQL · 2026
ShopEasy — E-Commerce Platform
A complete e-commerce web application built from scratch — with user authentication, product browsing, cart management, and order placement. Handled everything from database design to frontend UI.
  • Built a custom REST API server from scratch using Python's built-in http.server and BaseHTTPRequestHandler — no framework used
  • Implemented token-based authentication with SHA-256 password hashing; token stored in localStorage and validated on every protected API request
  • Designed a normalized MySQL database with 6 tables — users, categories, products, cart, orders, order_items — with foreign key relationships
  • Connected MySQL using PyMySQL with DictCursor for efficient JSON output of query results
  • Built React frontend with React Router, Axios interceptors for automatic token attachment, and useState/useEffect hooks
  • Implemented live search and category filter on frontend using JavaScript array filtering — no extra API calls needed
  • Handled CORS manually by adding Access-Control headers in Python server for React-to-Python communication
★ Built the REST API in pure Python without any framework — this means I understand exactly how HTTP methods (GET, POST, PUT, DELETE) and request-response cycles work at the core level.
